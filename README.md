# Employee Management System

A modern Employee Management System built with **React.js**, **Vite**, **Tailwind CSS**, and **Local Storage**. This project allows an admin to manage employees and assign tasks, while employees can view and update their assigned tasks.

## 🚀 Live Demo

https://employee-management-system-bay-five.vercel.app/

## 📂 GitHub Repository

https://github.com/shubha976625/employee-management-system

---

## ✨ Features

### Admin
- Admin Login
- Create New Tasks
- Assign Tasks to Employees
- View Employee Task Status
- Dashboard with Task Statistics

### Employee
- Employee Login
- View Assigned Tasks
- Accept Tasks
- Mark Tasks as Completed
- Mark Tasks as Failed

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/shubha976625/employee-management-system.git
```

Move into the project

```bash
cd employee-management-system
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```
